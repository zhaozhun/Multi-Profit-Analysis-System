# 数据更新与利润指标设计

## 1. 背景

用户需要：
1. 按照新方案更新数据（关联维度、费用分摊等）
2. 设计月度/季度/年度利润指标
3. 把这些指标拆分到指标库

## 2. 当前数据状态

- biz_ledger：969,840条数据（2024-01 到 2026-06）
- dimension_master：116条维度数据
- atomic_indicator：16条原子指标
- cost_type_config：12条费用类型配置

## 3. 数据更新方案

### 3.1 维度关联

biz_ledger 已有维度字段（org_id、product_id、biz_line_id、dept_id、channel_id、manager_id），需要确保这些字段与 dimension_master 表关联正确。

**更新内容**：
1. 检查并修复维度ID关联
2. 确保所有维度数据存在

### 3.2 费用分摊

**分摊逻辑**：
1. 获取每月费用原始数据
2. 按分摊规则计算分摊比例
3. 将分摊结果写入 cost_allocation_result
4. 更新 biz_ledger.op_cost

### 3.3 利润计算

**利润公式**：
- 净利润 = 利息收入 - FTP成本 - 风险成本 - 运营成本
- 资产利润 = 利息收入 - FTP成本 - 风险成本 - 运营成本
- 负债利润 = FTP收入 - 对客利息支出 - 运营成本

## 4. 利润指标设计

### 4.1 指标分类

**月度指标**：
- MONTHLY_NET_PROFIT：月度净利润
- MONTHLY_ASSET_PROFIT：月度资产利润
- MONTHLY_LIABILITY_PROFIT：月度负债利润
- MONTHLY_INTEREST_INCOME：月度利息收入
- MONTHLY_FTP_COST：月度FTP成本
- MONTHLY_RISK_COST：月度风险成本
- MONTHLY_OP_COST：月度运营成本

**季度指标**：
- QUARTERLY_NET_PROFIT：季度净利润
- QUARTERLY_ASSET_PROFIT：季度资产利润
- QUARTERLY_LIABILITY_PROFIT：季度负债利润

**年度指标**：
- YEARLY_NET_PROFIT：年度净利润
- YEARLY_ASSET_PROFIT：年度资产利润
- YEARLY_LIABILITY_PROFIT：年度负债利润

### 4.2 指标计算公式

**月度净利润**：
```sql
SELECT SUM(net_profit) FROM biz_ledger WHERE account_period = '2026-01'
```

**季度净利润**：
```sql
SELECT SUM(net_profit) FROM biz_ledger WHERE account_period BETWEEN '2026-01' AND '2026-03'
```

**年度净利润**：
```sql
SELECT SUM(net_profit) FROM biz_ledger WHERE account_period LIKE '2026%'
```

## 5. 指标库拆分

### 5.1 原子指标（atomic_indicator）

存储基础计算指标，直接从 biz_ledger 表聚合。

### 5.2 派生指标（derived_indicator）

存储复合计算指标，基于原子指标计算。

### 5.3 指标汇总（indicator_summary）

存储预计算的指标结果，用于快速查询。
