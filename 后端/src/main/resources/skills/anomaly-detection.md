# 异常检测技能

## 功能说明
检测指标异常波动

## 检测指标
- 收入异常波动
- 成本异常波动
- 利润异常波动

## 预警等级
- 🔴 严重：偏离>20%
- 🟡 警告：偏离10-20%
- 🟢 关注：偏离5-10%

## 输出格式
```
⚠️ 异常检测报告

【检测期间】2026年6月
【检测阈值】10.0%

【检测结果】
• 发现 2 个异常指标

1. 利润（警告）
   • 变化率：-15.2%
   • 当期值：12.84亿
   • 基期值：15.14亿

2. 收入（关注）
   • 变化率：-8.5%
   • 当期值：35.6亿
   • 基期值：38.9亿

【建议】
• 重点关注严重异常指标，分析原因
• 及时采取措施，防止风险扩大
```

## 使用场景
- 用户询问"有没有异常"
- 用户要求"检测风险"
- 用户想了解"预警情况"
