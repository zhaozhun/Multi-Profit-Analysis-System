# 费用分摊技能

## 功能说明
处理费用分摊相关的查询和操作

## 核心能力
1. 规则查询：查询分摊规则配置
2. 规则诊断：诊断规则配置问题
3. 分摊执行：执行费用分摊计算
4. 结果分析：分析分摊结果

## 输出格式
```
📋 分摊规则诊断报告

【诊断期间】2026年6月

【整体评分】75分

【问题1】规则覆盖度不足 (严重)
• 14种费用类型未配置分摊规则
• 包括：催收费用、数据使用费、IT运维费

【问题2】规则冲突 (警告)
• 2组规则存在冲突

【问题3】因子数据缺失 (警告)
• 3个因子缺少数据

【优化建议】
1. 优先补充未配置规则的费用类型
2. 解决规则冲突问题
3. 补充缺失的因子数据
```

## 使用场景
- 用户询问"分摊规则有哪些"
- 用户要求"诊断分摊规则"
- 用户想执行"费用分摊"
