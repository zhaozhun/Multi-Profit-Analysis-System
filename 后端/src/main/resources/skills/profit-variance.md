# 利润波动归因技能

## 功能说明
分析利润变化的驱动因素，找出关键影响维度

## 分析方法
1. 获取当期和基期整体数据
2. 按机构维度计算贡献度
3. 按产品维度计算贡献度
4. 生成归因分析报告

## 输出格式
```
📊 利润波动归因分析

【整体情况】
• 分析期间：2026-05 → 2026-06
• 当期利润：12.84亿
• 基期利润：12.18亿
• 利润变化：0.66亿（5.42%）

【机构维度分析】
• 北京分行：贡献度 35.2%，变化 0.23亿
• 上海分行：贡献度 28.1%，变化 0.19亿
• 深圳分行：贡献度 18.5%，变化 0.12亿

【产品维度分析】
• 对公贷款：贡献度 42.3%，变化 0.28亿
• 零售贷款：贡献度 31.2%，变化 0.21亿
• 存款业务：贡献度 26.5%，变化 0.17亿

【结论】
• 利润增长主要来自北京分行影响
```

- 用户询问"为什么利润下降"
