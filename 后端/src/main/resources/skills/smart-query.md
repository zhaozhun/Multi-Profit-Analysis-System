# 智能查询技能

## 功能说明
处理简单的数据查询请求

## 查询类型
- 汇总查询：查看整体数据
- 排名查询：查看排名情况
- 对比查询：对比不同维度
- 趋势查询：查看数据趋势

## 输出格式
```
📊 查询结果

【查询维度】机构
【查询指标】利润
【查询期间】2026年6月

【数据结果】
1. 北京分行：2,850万
2. 上海分行：2,580万
3. 深圳分行：2,210万
4. 广州分行：1,960万
5. 杭州分行：1,820万
```

## 使用场景
- 用户询问"本月收入多少"
- 用户要求"各机构利润排名"
- 用户想了解"产品对比情况"
