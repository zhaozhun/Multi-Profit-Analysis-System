# 敏感性分析技能

## 功能说明
分析各因素对盈利的影响程度

## 分析因素
- 收入变化对利润的影响
- 成本变化对利润的影响
- 风险成本变化对利润的影响
- 运营成本变化对利润的影响

## 输出格式
```
📊 敏感性分析报告

【分析期间】2026年6月
【变化范围】10.0%

【分析结果】
1. 收入
   • 敏感性系数：0.4000
   • 影响程度：高敏感

2. 成本
   • 敏感性系数：0.3000
   • 影响程度：中敏感

3. 风险成本
   • 敏感性系数：0.2000
   • 影响程度：中敏感

4. 运营成本
   • 敏感性系数：0.1000
   • 影响程度：低敏感

【结论】
• 最敏感因素：收入
• 建议重点关注该因素的变化
```

## 使用场景
- 用户询问"如果收入变化会怎样"
- 用户要求"分析影响因素"
- 用户想做"情景分析"
