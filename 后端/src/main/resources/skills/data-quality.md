# 数据质量扫描技能

## 功能说明
扫描数据质量，识别问题

## 检查维度
- 完整性：数据是否完整
- 一致性：数据是否一致
- 准确性：数据是否准确

## 输出格式
```
📊 数据质量扫描报告

【扫描期间】2026年6月
【扫描维度】机构

【整体评分】85分

【完整性】90分
• 良好，存在少量问题

【一致性】80分
• 良好，存在少量问题

【准确性】85分
• 良好，存在少量问题

【问题清单】
1. 🟡 数据完整性不足：90%
2. 🟡 数据一致性问题：80%
```

## 使用场景
- 用户询问"数据质量如何"
- 用户要求"检查数据质量"
- 用户想了解"数据校验结果"
