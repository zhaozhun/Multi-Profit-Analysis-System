# 变更日志

本文件记录项目的所有重要变更。格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)。

## [v1.0.0] - 2026-06-17

### 新增
- 经营驾驶舱：KPI指标卡、利润瀑布图、趋势监控、维度概览
- 6大维度分析：机构/条线/部门/产品/渠道/客户经理维度
- 指标管理：指标库管理、预计算功能
- 主数据管理：维度主数据维护
- AI智能助手：Claude AI集成，自然语言问答、经营简报生成
- 报表中心：台账报表、利润报表、自定义报表、AI报表
- 数据治理：数据质量监控

### 技术架构
- 后端：Spring Boot 3.2 + MyBatis + MySQL 8.0
- 前端：React 18 + TypeScript + Ant Design 5 + ECharts 5
- AI集成：Claude API (LangChain4j)
- 数据库：星型维度建模，11张核心表

### 部署
- 火山云服务器部署
- 后端8080端口，前端3000端口
- 30万条业务测试数据
