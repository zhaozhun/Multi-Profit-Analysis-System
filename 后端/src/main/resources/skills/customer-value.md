# 客户价值分析技能

## 功能说明
分析客户价值分布，找出高价值客户

## 分析维度
- 高价值客户：贡献80%利润的客户
- 中价值客户：贡献15%利润的客户
- 低价值客户：贡献5%利润的客户

## 输出格式
```
👥 客户价值分析报告

【分析期间】2026年6月

【整体情况】
• 客户总数：1,250
• 总利润：12.84亿

【高价值客户】TOP 125（贡献80%利润）
• 利润贡献：10.27亿
• 占比：80%
• 客户A：2,850万
• 客户B：2,580万
• 客户C：2,210万

【中价值客户】375个（贡献15%利润）
• 利润贡献：1.93亿
• 占比：15%

【低价值客户】750个（贡献5%利润）
• 利润贡献：0.64亿
• 占比：5%

【建议】
• 聚焦高价值客户维护，提升客户满意度
• 挖掘中价值客户潜力，提升客户等级
• 评估低价值客户，优化资源配置
```

## 使用场景
- 用户询问"哪些客户贡献大"
- 用户要求"分析客户价值"
- 用户想了解"客户分层情况"
