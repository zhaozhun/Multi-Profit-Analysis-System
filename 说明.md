# 多维盈利分析系统

基于 React + Spring Boot + Claude AI 的银行多维盈利分析系统。

## 核心功能

- 📊 **经营驾驶舱** - KPI指标卡、利润瀑布图、趋势监控、维度概览
- 📈 **6大维度分析** - 机构/条线/部门/产品/渠道/客户经理维度盈利分析
- 🤖 **AI智能助手** - 自然语言问答、自动经营简报、异常检测
- ✅ **数据校验** - 利润公式校验、环比异常检测、AI根因分析

## 技术栈

| 层级 | 技术 |
|------|------|
| 前端 | React 18 + TypeScript + Ant Design 5 + ECharts 5 |
| 后端 | Java 17 + Spring Boot 3 + MyBatis Plus |
| 数据库 | MySQL 8.0 (配置) + ClickHouse (分析) |
| AI | Claude API (LangChain4j) |
| 缓存 | Redis |

## 快速开始

### 1. 环境准备

```bash
# 安装依赖
- JDK 17+
- Node.js 18+
- MySQL 8.0
- ClickHouse (可选，可用MySQL替代)
- Redis
```

### 2. 数据库初始化

```bash
# MySQL
mysql -u root -p < backend/src/main/resources/schema.sql

# 导入Mock数据到MySQL（维度主数据）
mysql -u root -p multi_profit < backend/src/main/resources/data-mock.sql

# ClickHouse（可选）
clickhouse-client --multiquery < backend/src/main/resources/schema.sql
# 导入业务明细数据
python3 backend/src/main/resources/mock-data-generator.py
clickhouse-client --multiquery --format_csv_with_names=1 \
  --query "INSERT INTO multi_profit.biz_ledger FORMAT CSVWithNames" \
  < backend/src/main/resources/biz_ledger_mock.csv
```

### 3. 配置AI（可选）

```bash
export ANTHROPIC_API_KEY=sk-ant-xxxxx
```

### 4. 启动后端

```bash
cd backend
mvn spring-boot:run
```

### 5. 启动前端

```bash
cd frontend
npm install
npm run dev
```

访问 http://localhost:3000

## API接口

| 接口 | 说明 |
|------|------|
| GET /api/dashboard/overview | 驾驶舱全量数据 |
| GET /api/dashboard/waterfall | 瀑布图数据 |
| GET /api/dashboard/trend | 趋势数据 |
| GET /api/dimension/{dimType}/analysis | 维度分析数据 |
| POST /api/ai/chat | AI问答 |
| POST /api/ai/brief | 生成经营简报 |
| POST /api/validation/detect | 异常检测 |

## 核心计算逻辑

```
净利润 = 业务收入 - FTP资金成本 - 风险成本 - 运营成本
```
