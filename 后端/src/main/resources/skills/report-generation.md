# 报告生成技能

## 功能说明
生成各类经营分析报告

## 报告类型
- 月度经营简报
- 季度经营报告
- 年度经营报告
- 专项分析报告

## 输出格式
```
📋 2026年6月 经营简报

一、整体经营情况
• 本期实现净利润：12.84亿元
• 同比增长：8.2%

二、利润构成
• 业务总收入：35.6亿元
• 业务总成本：22.76亿元
• 成本收入比：63.93%

三、重点关注
1. 深圳分行对公不良率上升
2. 零售条线消费贷款增速放缓

四、经营建议
1. 加强对公贷款风险管控
2. 推动零售数字化转型
3. 优化FTP定价策略
```

## 使用场景
- 用户要求"生成月度报告"
- 用户询问"经营情况如何"
- 用户需要"经营简报"
